## About

All demos were built with **vibe coding** using [Claude Code](https://claude.ai/code) — Anthropic's AI coding assistant.
No demo code was written by hand; the entire implementation was generated through natural language conversation with Claude.

This project represents an initial effort to explore how AI can enhance the teaching and learning experience. I believe we are at a pivotal moment to rethink how we leverage AI tools to create more impactful educational content.
